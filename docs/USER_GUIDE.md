## 시작하기

### 첫 방문

1. 웹사이트에 접속하면 온보딩 프로세스가 시작됩니다.
2. 선호하는 콘텐츠 타입과 톤 스타일을 선택하세요.
3. 선택한 내용은 자동으로 기본값으로 설정됩니다.

### 계정 생성

1. 우측 상단의 "로그인" 버튼을 클릭하세요.
2. 이메일과 비밀번호로 회원가입하거나 소셜 로그인(Google, GitHub)을 사용하세요.
3. 회원가입 후 자동으로 로그인됩니다.

## 기본 기능

### 텍스트 프롬프트 생성

1. **텍스트 콘텐츠** 탭을 선택하세요.
2. **가이드 모드** 또는 **고급 모드**를 선택하세요.
   - 가이드 모드: 단계별 안내를 따라 프롬프트를 완성
   - 고급 모드: 모든 옵션을 한 번에 설정

3. **콘텐츠 유형**을 선택하세요:
   - 블로그 콘텐츠
   - LinkedIn, Facebook, Instagram
   - YouTube
   - X (Twitter), Threads, TikTok
   - Podcast, Email, Newsletter
   - Product Description, FAQ

4. **콘텐츠 목표**를 선택하세요:
   - 브랜드 인지도 강화
   - 전환/구매 유도
   - 참여/소통 활성화
   - 교육/인사이트 제공
   - 등등

5. **타겟 독자**를 설정하세요:
   - 나이, 성별, 직업

6. **톤 스타일**을 선택하세요:
   - 전문적, 친근한, 캐주얼, 격식 있는 등

7. **자연어 프롬프트**를 입력하세요:
   - 원하는 내용을 자유롭게 작성

8. **프롬프트 생성** 버튼을 클릭하세요.

### 이미지 프롬프트 생성

1. **이미지 생성** 탭을 선택하세요.
2. 원하는 이미지에 대한 설명을 입력하세요.
3. 스타일, 구도, 조명 등을 설정하세요.
4. 생성 버튼을 클릭하세요.

### 동영상 프롬프트 생성

1. **동영상 생성** 탭을 선택하세요.
2. 시나리오를 작성하세요.
3. 카메라 움직임, 샷 타입 등을 설정하세요.
4. 생성 버튼을 클릭하세요.

## 고급 기능

### 실시간 프리뷰

- 프롬프트를 입력하는 동안 실시간으로 미리보기를 확인할 수 있습니다.
- 여러 변형(짧은 버전, 긴 버전, 공식적, 비공식적)을 비교할 수 있습니다.

### A/B 테스트

- 같은 입력으로 여러 버전의 프롬프트를 생성하고 비교할 수 있습니다.
- 각 버전의 성과를 추적하여 최적의 프롬프트를 찾을 수 있습니다.

### 스마트 추천

- 사용 패턴을 학습하여 맞춤형 추천을 제공합니다.
- 유사한 프롬프트를 자동으로 찾아 제안합니다.

### 프롬프트 히스토리

- 생성한 모든 프롬프트가 자동으로 저장됩니다.
- 즐겨찾기, 태그, 검색 기능으로 쉽게 관리할 수 있습니다.

### 사용 통계

- 생성한 프롬프트 수, 카테고리별 분포 등을 확인할 수 있습니다.
- 사용 패턴을 분석하여 개선점을 찾을 수 있습니다.

## 협업 기능

### 워크스페이스

1. **워크스페이스** 메뉴에서 새 워크스페이스를 생성하세요.
2. 팀 멤버를 초대하세요.
3. 역할을 설정하세요:
   - **관리자**: 모든 권한
   - **멤버**: 프롬프트 생성 및 편집
   - **뷰어**: 조회만 가능

### 댓글 및 피드백

- 프롬프트에 댓글을 추가하여 팀과 소통할 수 있습니다.
- 답글 기능으로 토론을 이어갈 수 있습니다.

### 프롬프트 공유

- 프롬프트를 워크스페이스나 특정 사용자와 공유할 수 있습니다.
- 공개/비공개 설정이 가능합니다.

## FAQ

### Q: 프롬프트를 저장할 수 있나요?

A: 네, 모든 프롬프트는 자동으로 저장됩니다. 히스토리에서 확인할 수 있습니다.

### Q: 무료 플랜의 제한사항은?

A: 무료 플랜은 월 50개의 프롬프트 생성이 가능하며, 분당 10회의 API 요청이 가능합니다.

### Q: API 키는 어떻게 발급받나요?

A: 사용자 설정에서 API 키를 생성할 수 있습니다. API 키는 티어에 따라 요청 제한이 다릅니다.

### Q: 프롬프트를 수정할 수 있나요?

A: 네, 생성된 프롬프트는 언제든지 수정할 수 있으며, 버전 관리가 자동으로 됩니다.

### Q: 여러 AI 모델을 비교할 수 있나요?

A: 네, 멀티 모델 생성 기능을 사용하여 여러 AI 모델의 결과를 동시에 비교할 수 있습니다.
