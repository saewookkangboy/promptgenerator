# 프롬프트 생성기

자연어 텍스트를 입력하면 다양한 소셜 미디어 및 콘텐츠 플랫폼용 프롬프트를 생성하는 웹 애플리케이션입니다.

## 기능

- **다양한 콘텐츠 타입 지원**
  - 블로그 콘텐츠 (SEO, GEO, AEO 최적화)
  - 링크드인 뉴스피드
  - 페이스북 뉴스피드
  - 인스타그램 뉴스피드
  - 유튜브 영상 제목 및 설명

- **프롬프트 생성**
  - 메타 프롬프트: 콘텐츠 작성을 위한 기본 프롬프트
  - 컨텍스트 프롬프트: 콘텐츠 작성 가이드라인 및 구조
  - 해시태그: 최대 10개의 관련 해시태그 자동 생성

- **사용자 친화적 UI**
  - 모던하고 반응형 디자인
  - 원클릭 복사 기능
  - 직관적인 인터페이스

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 사용 방법

1. 콘텐츠 유형을 선택합니다 (블로그, 링크드인, 페이스북, 인스타그램, 유튜브)
2. 자연어로 프롬프트를 입력합니다
3. "프롬프트 생성하기" 버튼을 클릭합니다
4. 생성된 메타 프롬프트, 컨텍스트 프롬프트, 해시태그를 확인하고 복사합니다

## 기술 스택

- React 18
- TypeScript
- Vite
- CSS3
