# 프롬프트 생성기 (Prompt Generator)

자연어 텍스트를 입력하면 텍스트, 이미지, 동영상 생성용 프롬프트를 생성하는 통합 웹 애플리케이션입니다.

## 주요 기능

### 🎯 4가지 프롬프트 생성 카테고리

1. **텍스트 콘텐츠** - 소셜 미디어 및 블로그용 텍스트 프롬프트
2. **이미지 생성** - AI 이미지 생성 모델용 프롬프트
3. **동영상 생성** - AI 동영상 생성 모델용 프롬프트
4. **프롬프트 엔지니어링** - 고급 프롬프트 최적화 및 개선

### 🌐 Native English 변환

모든 프롬프트에 대해 자연스러운 영어 버전을 자동 생성하여 제공합니다.

## 기능 상세

### Phase 1: 텍스트 콘텐츠 프롬프트 생성 (초기 버전)

#### 지원 콘텐츠 타입
- **블로그 콘텐츠**: SEO, GEO, AEO 최적화된 2000~3000자 텍스트
- **링크드인 뉴스피드**: 전문적이고 비즈니스 중심의 콘텐츠
- **페이스북 뉴스피드**: 친근하고 공유하기 좋은 콘텐츠
- **인스타그램 뉴스피드**: 시각적이고 간결한 콘텐츠
- **유튜브 영상**: 검색 최적화된 제목과 상세한 설명

#### 상세 옵션 설정
- **타겟 독자 설정**
  - 나이: 10대, 20대, 30대, 40대, 50대, 60대 이상
  - 성별: 남성, 여성, 무관
  - 직업: 학생, 직장인, 개발자/프로그래머, 디자이너, 마케터, 기획자, 경영진/CEO, 자영업자, 프리랜서, 전문직, 기타
- **대화체 사용**: 체크박스로 선택 가능

#### 생성 결과
- **메타 프롬프트**: 콘텐츠 작성을 위한 기본 프롬프트
- **컨텍스트 프롬프트**: 콘텐츠 작성 가이드라인 및 구조 요구사항
- **해시태그**: 최대 10개의 관련 해시태그 자동 생성

### Phase 2: 이미지 생성 프롬프트 생성

#### 지원 모델
- **Midjourney**: 버전 4/5/6, Chaos, Seed, Style 설정 지원
- **DALL-E 3**: Vivid/Natural 스타일, 다양한 크기 지원
- **Stable Diffusion**: CFG Scale, Steps, Sampler 설정
- **Google Imagen**: 세밀한 스타일 제어
- **Adobe Firefly**: 상업적 사용 가능

#### 이미지 프롬프트 구성 요소
- **주제 (Subject)**: 이미지의 주요 내용
- **아트 스타일**: 사진/리얼리즘, 일러스트, 3D 렌더링, 수채화, 유화, 디지털 아트, 사진, 커스텀
- **구도 설정**
  - 프레이밍: 클로즈업, 미디엄 샷, 와이드 샷, 익스트림 와이드, 더치 앵글
  - 구도 법칙: 3분할 법칙, 황금비, 대칭, 중앙 배치, 다이나믹
  - 원근감: 눈높이, 조감도, 웜아이, 공중
- **조명 설정**
  - 타입: 자연광, 스튜디오, 드라마틱, 부드러운, 강한, 림 라이트, 역광
  - 방향: 앞, 옆, 뒤, 위, 아래
  - 강도: 0-100 슬라이더
  - 색감: 따뜻한, 차가운, 중립적, 골든아워, 블루아워
- **색상 팔레트**
  - 분위기: 생생한, 차분한, 단색, 파스텔, 고대비, 따뜻한, 차가운
  - 조화: 보색, 유사색, 삼원색, 단색조
- **기술적 설정**
  - 아스펙트 비율: 1:1, 4:3, 16:9, 21:9, 9:16, 커스텀
  - 품질: 1-5 단계
  - 해상도: 512x512 ~ 4096x4096
- **네거티브 프롬프트**: 제외할 요소 입력

#### 생성 결과
- **메타 프롬프트**: 이미지 생성용 기본 프롬프트
- **컨텍스트 프롬프트**: 이미지 생성 컨텍스트 및 설정 정보
- **전체 프롬프트**: 모델별 파라미터 포함 완전한 프롬프트 (복사용)
- **해시태그**: 이미지 관련 해시태그

### Phase 3: 동영상 생성 프롬프트 생성

#### 지원 모델
- **OpenAI Sora 2**: 고품질, 긴 동영상 (최대 60초), 고해상도
- **Google Veo 3**: 초고해상도, 긴 컷, 세밀한 제어
- **Runway Gen-3**: 영화적 품질
- **Pika Labs**: 실시간 동영상 생성
- **Stable Video Diffusion**: 오픈소스 동영상 생성

#### 동영상 프롬프트 구성 요소
- **전체 스타일 설정**
  - 장르: 액션, 드라마, 코미디, 공포, SF, 다큐멘터리, 뮤직비디오
  - 분위기: 긴장감, 평화로움, 역동적, 우울한, 에너지틱, 신비로운
  - 색감: 따뜻한, 차가운, 고대비, 저채도, 생생한, 단색
  - 영화적 품질: 체크박스 옵션
- **문맥적 톤앤매너**: 서정적, 서사적, 서술적, 대화적 등
- **정성적 톤앤매너**: 따뜻한, 차가운, 부드러운, 강렬한, 우아한 등
- **참조 이미지 설정**
  - 참조 이미지 유무 체크박스
  - 참조 이미지 설명 입력
- **스토리보드 기능**
  - 여러 장면 추가/삭제
  - 각 장면별 설정:
    - 장면 설명
    - 길이 (초)
    - 카메라 움직임: 정적, 돌리, 팬, 틸트, 줌, 트래킹, 오빗, 크레인
    - 샷 타입: 익스트림 클로즈업, 클로즈업, 미디엄, 와이드, 익스트림 와이드, 이스타블리싱
    - 카메라 앵글: 눈높이, 하이앵글, 로우앵글, 조감도, 더치
    - 모션 속도: 슬로우 모션, 일반, 빠른, 타임랩스
- **기술적 설정**
  - 프레임레이트: 24fps, 30fps, 60fps
  - 해상도: 720p, 1080p, 4K, 8K
  - 아스펙트 비율: 16:9, 21:9, 1:1, 9:16
  - 총 길이: 5-120초 (슬라이더)

#### 자동 상세 묘사 기능
- **장르별 자동 묘사**: 선택한 장르에 맞는 상세한 묘사 자동 추가
  - 액션: "dynamic action, intense movement, high energy"
  - 드라마: "emotional depth, character-driven, nuanced performance"
  - 코미디: "light-hearted, humorous, playful atmosphere"
  - 공포: "suspenseful, eerie, atmospheric tension"
  - SF: "futuristic, technological, otherworldly"
  - 다큐멘터리: "realistic, authentic, observational"
  - 뮤직비디오: "rhythmic, stylized, visually striking"
- **분위기별 자동 묘사**: 선택한 분위기에 맞는 상세한 묘사 자동 추가
  - 긴장감: "building tension, suspenseful pacing, dramatic moments"
  - 평화로움: "tranquil atmosphere, gentle pacing, serene mood"
  - 역동적: "energetic movement, fast-paced, vibrant energy"
  - 우울한: "somber tone, reflective mood, emotional depth"
  - 에너지틱: "high energy, fast-paced, vibrant"
  - 신비로운: "enigmatic atmosphere, subtle hints, intriguing"

#### 생성 결과
- **메타 프롬프트**: 동영상 생성용 전체 프롬프트
- **컨텍스트 프롬프트**: 동영상 생성 컨텍스트 및 설정 정보
- **전체 프롬프트**: 모든 장면을 연결한 완전한 프롬프트
- **장면별 프롬프트**: 각 장면의 개별 프롬프트 (개별 복사 가능)
- **해시태그**: 동영상 관련 해시태그

### Phase 4: 프롬프트 엔지니어링

#### 지원 방법
1. **Zero-shot Learning**: 예시 없이 바로 실행 가능한 프롬프트
2. **Chain of Thought (CoT)**: 단계별 사고 과정을 유도하는 프롬프트
   - 사용자 정의 단계 추가/삭제
   - 추론 과정 명시 옵션
3. **Few-shot Learning**: 예시를 포함한 학습 프롬프트
   - 여러 예시 추가/삭제
   - 각 예시별 입력/출력/설명 입력
4. **Role-based Prompting**: 역할 기반 프롬프트
   - 역할 입력 (예: 전문 마케팅 컨설턴트)
   - 전문성 목록 추가/삭제
   - 관점 설정 (선택사항)
5. **Optimize**: 프롬프트 자동 최적화
   - 명확성 개선
   - 구조 개선
   - 키워드 강조
   - 목표 길이 설정 (짧게/보통/길게)

#### 최적화 기능
- **프롬프트 점수화**: 효과성 예측 점수 (0-100)
- **자동 개선 제안**: AI가 프롬프트를 분석하고 개선안 제시
- **개선 사항 표시**: 적용된 개선 사항 목록
- **제안 사항 표시**: 추가 개선을 위한 제안 목록

#### 생성 결과
- **메타 프롬프트**: 엔지니어링된 프롬프트
- **컨텍스트 프롬프트**: 엔지니어링 컨텍스트 및 설정 정보
- **전체 프롬프트**: 최종 최적화된 프롬프트
- **최적화 결과** (Optimize 방법 사용 시):
  - 원본 프롬프트
  - 최적화된 프롬프트
  - 점수 (0-100)
  - 개선 사항 목록
  - 제안 사항 목록

### Native English 변환 기능

모든 프롬프트 생성기에 공통으로 적용된 기능입니다.

#### 주요 특징
- **구조화된 영어 템플릿**: 각 생성기별로 최적화된 영어 프롬프트 템플릿 사용
- **자동 변환**: 프롬프트 생성 시 영문 버전 자동 생성
- **실시간 토글**: 한국어 ↔ English 즉시 전환
- **Native English**: 자연스러운 영어 표현으로 최적화
- **AI 최적화**: AI 모델에 적합한 표현 사용

#### 변환 방식
- **템플릿 기반 생성**: 각 카테고리별 전용 영어 템플릿 사용
  - 텍스트: 콘텐츠 타입별 영어 메타/컨텍스트 프롬프트
  - 이미지: 모델별 영어 프롬프트 구조
  - 동영상: 장면별 영어 프롬프트 및 전체 컨텍스트
  - 엔지니어링: 방법별 영어 프롬프트 최적화
- **자동 번역**: 템플릿이 없는 경우 한국어 표현을 자연스러운 영어로 변환
- **문장 구조 개선**: 한국어식 어순을 영어식으로 자동 변환
- **관용구 적용**: 자연스러운 영어 관용구 및 표현 사용

#### UI/UX
- 각 결과 카드에 "English" / "한국어" 토글 버튼
- 영어 버전 표시 시 "Native English Version" 표시
- 현재 표시 중인 언어의 프롬프트 복사 가능
- 동영상 생성 시 장면별로도 영어 버전 제공
  - 각 장면에 언어 토글 버튼
  - 장면별 영어 프롬프트 개별 복사 가능

  - CSS 애니메이션 최적화
  - 불필요한 리렌더링 방지

#### 사용자 인터페이스

  - 전체 밑줄 그리드 디자인
  - Hover 시 검정 배경 + 흰색 텍스트
- **접을 수 있는 상세 옵션**: 깔끔한 UI를 위한 접기/펼치기 기능
- **원클릭 복사**: 모든 프롬프트에 원클릭 복사 기능
- **로딩 상태 표시**: 프롬프트 생성 중 로딩 스피너 표시
- **에러 처리**: 사용자 친화적인 에러 메시지 표시
- **입력 검증**: 실시간 입력 검증 및 피드백
- **버튼 상태 관리**: 생성 중 버튼 비활성화 및 로딩 표시

## 설치 및 실행

### 요구사항
- Node.js 18 이상
- npm 또는 yarn

### 설치

```bash
# 저장소 클론
git clone https://github.com/saewookkangboy/promptgenerator.git
cd prompt-generator

# 의존성 설치
npm install
```

### 환경 변수 (.env)

프로젝트 루트에 `.env` 파일을 생성하고 아래 값을 추가하세요.   
민감한 값이므로 버전 관리에 포함하지 않습니다.

```bash
DATABASE_URL="postgresql://YOUR_DB_URL"
DEEPL_API_KEY="YOUR_DEEPL_API_KEY"
OPENAI_API_KEY="YOUR_OPENAI_API_KEY" # DeepL 요약 파이프라인 압축용 (선택)
OPENAI_SUMMARIZE_MODEL="gpt-4o-mini" # 선택, 기본값 gpt-4o-mini
```

### DeepL 호출 로그

- 서버는 DeepL 번역 요청/응답 요약을 `logs/deepl.log` 파일에 JSON Lines 형식으로 기록합니다.
- 로그에는 요청 수, 문자 수, 소요 시간, 응답 코드 등이 포함되어 추후 진단에 활용할 수 있습니다.

### 개발 서버 실행

```bash
npm run dev
```

## 사용 방법

### 텍스트 콘텐츠 생성

1. "텍스트 콘텐츠" 탭 선택
2. 콘텐츠 유형 선택 (블로그, LinkedIn, Facebook, Instagram, YouTube)
3. 자연어 프롬프트 입력
4. "상세 옵션" 펼치기 (선택사항)
   - 나이, 성별, 직업 선택
   - 대화체 사용 여부 선택
5. "프롬프트 생성하기" 클릭
6. 생성된 프롬프트 확인 및 복사
7. "English" 버튼으로 영문 버전 확인 (선택사항)

### 이미지 생성 프롬프트 생성

1. "이미지 생성" 탭 선택
2. 이미지 생성 모델 선택 (Midjourney, DALL-E 등)
3. 주제 입력
4. 아트 스타일 선택
5. 구도, 조명, 색상 설정
6. "고급 옵션" 펼치기 (선택사항)
   - 조명 강도 조절
   - 품질 설정
   - 네거티브 프롬프트 추가
7. "이미지 프롬프트 생성하기" 클릭
8. 생성된 프롬프트 확인 및 복사
9. "English" 버튼으로 영문 버전 확인 (선택사항)

### 동영상 생성 프롬프트 생성

1. "동영상 생성" 탭 선택
2. 동영상 생성 모델 선택 (Sora, Veo 등)
3. 전체 스타일 설정 (장르, 분위기, 색감)
4. 문맥적 톤앤매너 입력 (선택사항)
5. 정성적 톤앤매너 입력 (선택사항)
6. 참조 이미지 설정 (선택사항)
   - "참조 이미지 있음" 체크
   - 참조 이미지 설명 입력
7. 스토리보드 구성
   - "+ 장면 추가" 버튼으로 장면 추가
   - 각 장면별 설명, 길이, 카메라, 모션 설정
8. 기술 사양 설정 (프레임레이트, 해상도, 아스펙트 비율)
9. "동영상 프롬프트 생성하기" 클릭
10. 생성된 프롬프트 확인 및 복사
    - 전체 프롬프트 또는 장면별 프롬프트 복사 가능
    - 각 장면에서 "English" 버튼으로 영문 버전 확인 (선택사항)

### 프롬프트 엔지니어링

1. "프롬프트 엔지니어링" 탭 선택
2. 엔지니어링 방법 선택
   - Zero-shot: 기본 프롬프트만 입력
   - Chain of Thought: 단계별 사고 과정 설정
   - Few-shot: 예시 추가
   - Role-based: 역할 및 전문성 설정
   - Optimize: 최적화 옵션 설정
3. 기본 프롬프트 입력
4. 선택한 방법에 맞는 추가 설정 입력
5. "프롬프트 엔지니어링 생성하기" 클릭
6. 생성된 프롬프트 확인 및 복사
7. "English" 버튼으로 영문 버전 확인 (선택사항)

### Admin 대시보드 사용

1. 메인 페이지 헤더 우측의 "Admin" 버튼 클릭
2. 로그인 페이지에서 인증 정보 입력 (관리자에게 문의)
3. Admin 대시보드에서 통계 확인
   - 총 방문수
   - 총 생성 건수
   - 카테고리별 생성 건수 (텍스트, 이미지, 동영상, 엔지니어링)
4. 프롬프트 생성 기록 확인
   - 카테고리별 필터링 가능 (전체/텍스트/이미지/동영상/엔지니어링)
   - 기록 행 클릭 시 상세 옵션 확인
   - 각 카테고리별 선택된 옵션 테이블 표시:
     - 텍스트: 콘텐츠 타입, 나이, 성별, 직업, 대화체 여부
     - 이미지: 아트 스타일, 프레이밍, 조명, 색상 무드, 종횡비, 품질, 네거티브 프롬프트
     - 동영상: 장르, 무드, 총 길이, FPS, 해상도, 장면 수, 참조 이미지 여부
     - 엔지니어링: 엔지니어링 방법
5. "메인으로" 버튼으로 일반 모드 복귀
6. "로그아웃" 버튼으로 Admin 세션 종료

  - 성능 최적화 (useMemo, useCallback)
  - 코드 재사용성 향상 (공통 훅 및 유틸리티)
  - 타입 안정성 개선
  - 중앙화된 상수 관리
  - SEO 개선 (meta 태그 추가)

  - 로컬 스토리지 기반 인증 상태 관리

    - **엔지니어링**: 엔지니어링 방법
  - 옵션 정보를 테이블 형식으로 정리하여 표시
  - 선택된 기록의 전체 프롬프트 및 모델 정보 표시

  - 방문수 자동 증가
  - 통계 자동 업데이트
- **UI/UX**
  - 반응형 디자인 (모바일/태블릿/데스크톱)
  - 모던한 카드 디자인
  - 행 선택 시 시각적 피드백
  - "메인으로" 버튼으로 일반 모드 복귀
  - 접근성 고려 (키보드 네비게이션, 포커스 상태)
