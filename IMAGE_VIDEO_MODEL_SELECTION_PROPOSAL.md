# 이미지/동영상 생성 모델 선택 기능 제안서

## 📋 개요

가이드모드와 고급모드에서 이미지 생성 및 동영상 생성 모델을 선택할 수 있는 기능을 추가하고, 최신 모델로 업데이트하는 제안서입니다.

---

## 🎯 목표

1. **가이드모드**: 초보자도 쉽게 모델을 선택할 수 있도록 간단한 UI 제공
2. **고급모드**: 전문가가 다양한 모델과 상세 옵션을 선택할 수 있도록 제공
3. **최신 모델 반영**: 2025-2026년 최신 이미지/동영상 생성 모델 업데이트

## 🎨 UI/UX 설계

### 가이드모드 (Guide Mode)

#### 이미지 생성 모델 선택

**위치**: 가이드모드 1단계 (주제 입력 전 또는 후)

**UI 구성**:
```
┌─────────────────────────────────────┐
│ 생성 모델 선택 (선택사항)            │
│                                     │
│ ○ 추천 모델 (자동 선택)             │
│   └─ Flux.2 Pro - 빠르고 고품질     │
│                                     │
│ ○ 빠른 생성                         │
│   └─ Nano Banana Pro - 즉시 결과   │
│                                     │
│ ○ 고품질 사진                       │
│   └─ Imagen 3 - 사진 같은 리얼리즘 │
│                                     │
│ ○ 예술적 스타일                     │
│   └─ Midjourney v6 - 창의적 표현    │
└─────────────────────────────────────┘
```

**특징**:
- 모델 이름보다는 **용도 중심**으로 표시
- 각 옵션에 간단한 설명 제공
- 기본값: "추천 모델" (프롬프트 분석 후 자동 추천)
- 모델별 간단한 특징 아이콘 표시

## 🎯 추천 기능

### 1. AI 모델 추천

가이드모드에서 "추천 모델" 선택 시:
- 사용자 프롬프트 분석
- 스타일, 용도 파악
- 적합한 모델 자동 추천

**예시**:
- "텍스트가 포함된 포스터" → Ideogram 2.0 추천
- "사진 같은 리얼리즘" → Imagen 3 또는 Flux.2 Pro 추천
- "예술적 일러스트" → Midjourney v6 추천

### 2. 모델 비교 기능

고급모드에서:
- "모델 비교 보기" 버튼 클릭
- 모달 또는 사이드 패널로 비교 테이블 표시
- 해상도, 속도, 비용, 특징 비교

### 3. 샘플 미리보기

- 각 모델별 샘플 이미지/동영상 제공
- 동일한 프롬프트로 여러 모델 결과 비교
